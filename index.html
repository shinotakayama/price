Information and Price in Dynamic Financial Markets

Computer Codes for Equilibrium Strategies

This set of MATLAB codes computes equilibrium strategies for the paper: "Price Manipulation, Dynamic Informed Trading, and the Uniqueness of Equilibrium in Sequential Trading" (Journal of Economic Dynamic and Control, 2021).

Main Programs

  • mainprog.m - Solves the equilibrium
  • savere.m - Saves simulation results
  • jacHL.m - Debugging tool

Functions

  • nextvalue.m - Computes function values
  • ask.m - Computes ask price via Bayes' rule
  • bid.m - Computes bid price via Bayes' rule
  • fd.m - Finds interval for a given value
  • incentive.m - Incentive compatibility calculations
