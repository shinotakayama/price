Information and Price in Dynamic Financial Markets

Computer Codes for Equilibrium Strategies

This set of MATLAB codes computes equilibrium strategies for the paper: "Price Manipulation, Dynamic Informed Trading, and the Uniqueness of Equilibrium in Sequential Trading" (Journal of Economic Dynamic and Control, 2021).

Main Programs

  • mainprog.m - This program solves the equilibrium.
  • plotfig.m - Produces pictures from the result (**.mat**).
  • savere.m - Saves the simulation results.
  • jactry.m - Used to check for bugs when issues arise.

Functions

  • nextvalue.m - Computes the function value: `m(int) * (x - b(int)) + f(int)`.
  • ask.m - Computes ask price via Bayes' rule.
  • bid.m - Computes bid price via Bayes' rule.
  • fd.m - Finds the interval `[b(j), b(j+1))` to which value `f` belongs.
  • profits.m - Computes profit for each strategy given price and value functions.
  • incentive.m - Checks if honest strategy is incentive-compatible.

Trading & Strategy Functions

  • highbuy.m - Computes profit for High to buy.
  • highsell.m - Computes profit for High to sell.
  • lowbuy.m - Computes profit for Low to buy.
  • lowsell.m - Computes profit for Low to sell.

Equilibrium & Indifference Conditions

  • checkH.m - Determines if type H manipulates.
  • checkL.m - Determines if type L manipulates.
  • calaskH.m - Finds ask price for type H.
  • calbidH.m - Computes bid price from calaskH.m.
  • calaskL.m - Finds ask price for type L.
  • calbidL.m - Computes bid price from calaskL.m.

Mathematical Computation Functions

  • sol2real.m - Solves `axÂ² + bx + c = 0` for two solutions.
  • diffun.m - Computes profit differences between strategies.
  • fixedHL.m - Returns strategies for type H and L when both manipulate.
  • myfunHL.m - Computes approximated indifference conditions.
  • newtonHL.m - Uses Newton-Raphson method to find solutions.
  • value1.m - Computes today's values for type L and H.
  • valueL.m - Returns today's value for type L given bid price.
  • valueH.m - Returns today's value for type H given ask price.
