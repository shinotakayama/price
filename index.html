Information and Price in Dynamic Financial Markets

This is a set of computer codes that computes equilibrium strategies for Price manipulation, dynamic informed trading, and the uniqueness of equilibrium in sequential trading (Journal of Economic Dynamic and Control, 2021)

mainprog.m This program solves the equilibrium.

savere.m This program saves the simulation results.

jacHL.m This program is used to check the bug whenever any issue arises.

nextvalue.m nextvalue(x, int, b, m, f) computes the value of a function.

ask.m ask(b, u, r, hb, ls) computes ask price via Bayes' rule.

bid.m bid(b, u, r, hb, ls) computes bid price via Bayes' rule.

fd.m fd(x0, x1, b, f) finds the interval to which the value f belongs.

incentive.m Computes incentive compatibility.

highbuy.m Computes profit for High to buy.

highsell.m Computes profit for High to sell.

lowbuy.m Computes profit for Low to buy.

lowsell.m Computes profit for Low to sell.

checkH.m Determines whether type H manipulates.

checkL.m Determines whether type L manipulates.

calaskH.m Finds ask price for type H.

calbidH.m Computes bid price from calaskH.m.

calaskL.m Finds ask price for type L.

calbidL.m Computes bid price from calaskL.m.

sol2real.m Solves ax^2 + bx + c = 0 for two solutions.

diffun.m Computes profit differences between strategies.

fixedHL.m Returns strategies for type H and L when both manipulate.

myfunHL.m Computes approximated indifference conditions.

newtonHL.m Newton-Raphson method to find solutions.

value1.m Computes today's values for type L and H.

valueL.m Returns today's value for type L given bid price.

valueH.m Returns today's value for type H given ask price.
